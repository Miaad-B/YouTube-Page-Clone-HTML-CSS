# YouTube Page Clone (HTML & CSS)

This project is a **static clone of a YouTube video listing page**, created as part of an online **HTML & CSS course**. The main goal was to practice layout, positioning, and basic page structure by recreating a familiar interface.

## Overview

The page includes:
- Header with logo, search bar, icons and tooltips
- Sidebar navigation with common YouTube-style links
- Main content area displaying six video previews
- Each video preview shows thumbnail, title, channel name, views, and upload time

All `<a>` links in the project point to the **actual YouTube videos and profile pages** used as examples in the course.

## Technologies Used

- **HTML5** for page structure and semantic elements  
- **CSS3** for layout, styling, and basic responsiveness  
- Grid and Flex layouts were used

## Learning Focus

This project was built **step by step following the instructor’s guidance**. It helped me practice:

- Structuring a page with reusable sections (header, sidebar, main content)
- Using Flex and Grid layouts to create common UI layouts
- Styling text, icons, and thumbnails consistently
- Organizing CSS for readability and maintainability

## How to View

1. Clone or download the repository.  
2. Open `index.html` in your browser.
